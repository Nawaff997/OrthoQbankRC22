📄 How OrthoQbankRC2 Works
MCQs are sourced from the original Saudi Orthopedic Part 1 Qbank PDF (2008–2022) plus any additional PDFs you import. Tap any answer choice to reveal the correct answer with an explanation and reference. Answers are drawn from standard orthopedic teaching consistent with Orthobullets, AAOS, and AO Surgery Reference, with the specific reference topic cited. Use the 🔒 Import PDF button to add more Qbank PDFs (password required) — each question carries a source tag showing which file it came from, and you can filter by source. Imported questions are stored locally in your browser.
